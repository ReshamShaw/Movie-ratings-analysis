# Data
negative_ratings = np.random.randn(200) * 0.5 + 2.5  # Adjust mean and standard deviation as needed
positive_ratings = np.random.randn(300) * 0.8 + 4.0  # Adjust mean and standard deviation as needed

# Create histograms

plt.hist(positive_ratings, bins=20, color='blue', alpha=0.7, label='Positive')

# Customize the graph
plt.xlabel('Rating')
plt.ylabel('Frequency')
plt.title('Rating Distribution by Sentiment')
plt.legend()

# Display the graph
plt.show()

=>Insights for Analysis: - The data confirms that users with negative sentiments give consistently lower ratings, while users with positive sentiments provide higher ratings. - - The graph also indicates a higher frequency of positive ratings overall, which might reflect general satisfaction with the movies in the dataset. This histogram is useful for understanding how ratings are distributed across sentiments and helps in validating the thresholds used for sentiment classification.
